Toga

Une boîte à outils d'interface graphique native Python et OS.

Briefcase

Convertir un projet Python en une application native autonome.

Applications

Projets autonomes d'utilisateurs finaux réalisés à l'aide des outils BeeWare, notamment :

Podium

Un outil de présentation de diapositives basé sur le balisage

... and others

Utilitaires

Pièces utilisées par d'autres outils BeeWare qui peuvent être utiles par eux-mêmes, y compris :

Rubicon Objective-C

Une bibliothèque pour faire le lien entre Python et le langage Objective-C les temps d'exécution.

Python Apple Support

Un méta-package pour construire une version de Python qui peut être intégrée dans un projet macOS, iOS, tvOS ou watchOS.

... and others

BeeWare n'est pas un outil ou une bibliothèque unique - c'est une collection de projets qui peuvent être utilisés pour aider à développer, déboguer et lancer des logiciels Python. Chaque outil suit la philosophie Unix qui consiste à bien faire une seule chose. Chaque outil Chaque outil peut être utilisé de manière isolée ou enchaîné afin de fournir un ensemble riche ensemble d'outils de programmation.

Mais il ne s'agit pas que de mots d'ordre : il y a aussi un [grenier où nous les projets que nous ne maintenons plus] (/project/attic/).
