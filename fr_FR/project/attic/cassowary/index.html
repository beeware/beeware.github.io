Cassowary

Une implémentation purement python de l'algorithme de résolution de contraintes Cassowary de résolution de contraintes.

Une implémentation purement Python de l'algorithme de résolution de contraintes [Cassowary de Cassowary] (http://constraints.cs.washington.edu/cassowary/). Cassowary est l'algorithme qui constitue le cœur du mécanisme de présentation visuelle d'OS X et d'iOS. visuel d'OS X et d'iOS.

Ce projet n'est plus utilisé. Il a été développé à l'origine pour être utilisé dans Toga ; cependant, Toga utilise maintenant une mise en page basée sur les CSS, implémentée dans Colosseum.
