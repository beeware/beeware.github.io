VOC

Un transpileur qui convertit le bytecode Python en bytecode Java.

Le COV remplit deux fonctions principales :

  • Compile les fichiers sources Python 3.4+ en fichiers de classe Java, ce qui vous permet d'exécuter le code Python sur une JVM (y compris la VM d'Android). d'exécuter le code Python sur une JVM (y compris la VM d'Android).
  • Fournit une API pour vous permettre de créer par programme des fichiers de classe Java.

Ce n'est pas encore une implémentation complètement conforme à Python 3.4 - il y a quelques caractéristiques du langage (comme nonlocal et quelques fonctions intégrées). certaines fonctionnalités du langage (comme nonlocal et certaines fonctions intégrées) qui doivent encore être implémentées, et il n'y a qu'une implémentation de la l'implémentation de la bibliothèque standard. Cependant, il est possible de convertir programmes Python simples, et même d'écrire [des applications Android simples] (https://github.com/eliasdorneles/tictactoe-voc).

Pour tester VOC, parcourez le guide [Getting démarrage] (https://voc.readthedocs.io/en/latest/index.html), puis commencez par le commencez par le premier tutoriel.

Si vous souhaitez contribuer au développement du COV, nous avons un [guide pour les premiers contributeurs] (https://voc.readthedocs.io/en/latest/how-to/contribute-code.html).
