VOC

Un transpiler che converte il bytecode Python in bytecode Java.

Il VOC svolge due funzioni principali:

  • Compila i file sorgenti di Python 3.4+ in file di classe Java, permettendo di di eseguire codice Python su una JVM (compresa la VM di Android).
  • Fornisce un'API che consente di creare programmaticamente file di classe Java.

Non è ancora un'implementazione completamente conforme a Python 3.4 - ci sono alcune caratteristiche del linguaggio (come nonlocal e alcune funzioni integrate). alcune caratteristiche del linguaggio (come nonlocal e alcune funzioni integrate) che devono ancora essere implementate, e c'è solo un'implementazione standard. Tuttavia, è possibile convertire semplici programmi Python e persino scrivere semplici applicazioni Android] (). Android.
