Gestori di pacchetti

Come gestire i pacchetti installati

L'installazione di software sul computer può essere interessante. A volte è scaricare un file e poi installarlo da soli, oppure copiare i file in punti specifici del computer. specifici del computer.

Tuttavia, i gestori di pacchetti aiutano a semplificare questo processo consentendo di l'automazione dell'installazione del software.

Esistono diversi livelli di gestori di pacchetti: per il sistema operativo e uno specifico per i pacchetti Python.

MacOSX

Homebrew è lo standard per l'installazione di software sul Mac. Mac. È il sistema utilizzato quando si esegue il comando brew install. .

Linux

A seconda della famiglia di sistemi operativi in uso, si utilizzerà apt-get (per Debian e Ubuntu) o yum (per Red Hat e CentOS).

Python

pip è il modo in cui si può installare il software Python. L'esecuzione di pip install utilizza il Python Packaging Index, noto anche come "PyPI". PyPI è un repository centrale per il codice Python. Molti progetti BeeWare possono essere installati con pip.
