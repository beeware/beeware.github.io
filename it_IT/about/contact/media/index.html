Media

Are you from a media organization and want to write something about BeeWare? Here are some resources you may find helpful.

Breve descrizione

BeeWare è una raccolta di strumenti e librerie per aiutare a scrivere interfacce utente delle applicazioni in Python.

Descrizione lunga

BeeWare è una raccolta di strumenti e librerie per aiutare a scrivere interfacce utente delle applicazioni in Python. Consente agli sviluppatori di scrivere applicazioni per iPhone, Android, Linux, macOS e Windows che sono indistinguibili dalle applicazioni scritte in modo nativo.
