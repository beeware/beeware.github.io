مديرو الحزم

كيف يمكنك إدارة الحزم المثبتة

قد يكون تثبيت البرامج على حاسوبك أمراً مثيراً للاهتمام. في بعض الأحيان تنزيل ملف ثم تثبيته بنفسك، أو نسخ الملفات إلى أماكن محددة على حاسوبك.

ومع ذلك، يساعد مديرو الحزم في جعل هذه العملية أسهل من خلال السماح ب أتمتة تثبيت البرامج.

هناك مستويات مختلفة من مديري الحزم: لنظام التشغيل ومستوى نظام التشغيل، بالإضافة إلى مستوى خاص بحزم بايثون.

macOS

Homebrew هو المعيار لتثبيت البرامج على جهاز Mac الخاص بك. هذا هو النظام الذي يُستخدم إذا قمت بتشغيل أمر 'brew install' أمرًا.

لينكس

اعتمادًا على عائلة نظام التشغيل الذي تقوم بتشغيله، ستستخدم AP-get' (لديبيان وأوبونتو) أوyum' (لريد هات و CentOS).

بايثون

pip is the way you can install Python software. Running pip install uses the Python Package Index, also known as "PyPI". PyPI is a central repository for Python code. Many BeeWare projects can be installed using pip.
