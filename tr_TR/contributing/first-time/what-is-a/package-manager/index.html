Paket Yöneticileri

Yüklü paketlerinizi nasıl yönetebilirsiniz?

Bilgisayarınıza yazılım yüklemek ilginç olabilir. Bazen sen bir dosyayı indirmeniz ve ardından kendiniz yüklemeniz veya dosyaları bilgisayarınızdaki belirli yerler.

Bununla birlikte, paket yöneticileri aşağıdakilere izin vererek bu süreci kolaylaştırmaya yardımcı olur yazılım kurulumunun otomasyonu.

Farklı seviyelerde paket yöneticileri vardır: işletim sistemi için seviyesinin yanı sıra özellikle Python paketleri için bir tane.

macOS

Homebrew üzerinde yazılım yüklemek için standarttır Mac'iniz. Bu, bir brew install çalıştırdığınızda kullanılan sistemdir Komuta.

Linux

Hangi işletim sistemi ailesini çalıştırdığınıza bağlı olarak apt-get (Debian ve Ubuntu için) veya yum (Red Hat ve CentOS için).

Python

pip is the way you can install Python software. Running pip install uses the Python Package Index, also known as "PyPI". PyPI is a central repository for Python code. Many BeeWare projects can be installed using pip.
