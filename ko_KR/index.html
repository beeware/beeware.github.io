BeeWare

한 번만 작성하세요. 어디에나 배포하세요.

Python으로 앱을 작성하고 iOS, Android, Windows에 출시하세요, MacOS, Linux, 웹 및 tvOS에서 풍부한 네이티브 사용자 인터페이스를 사용하여 앱을 출시하세요. 여러 앱, 하나의 코드베이스로 모든 플랫폼에서 완전한 네이티브 사용자 경험 제공 플랫폼.

튜토리얼 수강하기 기여하고 싶습니다 기부하고 후원하세요!

One codebase. Multiple apps.

BeeWare를 사용하면 Python으로 앱을 작성하여 여러 플랫폼에 출시할 수 있습니다. 여러 프로그래밍 언어로 앱을 다시 작성할 필요가 없습니다. 빌드 도구, 환경, 호환성 등에 문제가 없다는 뜻입니다.

Python native tools...

Python은 초보자도 쉽게 접근할 수 있지만 전문가가 사용하면 강력한 성능을 발휘하는 매우 유능한 언어임이 입증되었습니다. 컴퓨터에게 무언가를 하라고 지시해야 하는 모든 곳에서 Python을 사용할 수 있어야 하지 않을까요? 그리고 도구가 C 바인딩에 잘 매핑되는 비트뿐만 아니라 언어로서 Python의 모든 기능을 활용**해야 하지 않을까요?

...on mobile & desktop...

최신 컴퓨팅은 80x25 콘솔 창에서 이루어지지 않습니다. 풍부한 사용자 인터페이스를 갖춘 스마트폰, 태블릿, 데스크톱 컴퓨터에서 이루어집니다. 이러한 모든 위치에서 Python을 사용할 수 있어야 하며, 이러한 플랫폼의 고유한 기능을 활용**할 수 있어야 하지 않을까요?

...behaving natively.

최종 사용자는 도구가 어떤 언어로 작성되었는지 신경 쓸 필요가 없어야 합니다. 그 시작은 완전히 네이티브 도구처럼 보이고 작동하는 것에서 시작됩니다. 네이티브 앱이 제공하는 방식으로 제공되는 네이티브 모양, 네이티브 동작. Python 도구가 네이티브 도구와 똑같이 잘 어울리지 않을 이유가 있을까요?

This is BeeWare.

이것이 바로 BeeWare가 제공하는 것입니다. 풍부한 네이티브 사용자 인터페이스로 Python 코드를 작성하는 데 도움이 되는 도구와 해당 코드를 **iOS, Android, macOS, Linux, Windows, tvOS 등에서 실행하는 데 필요한 라이브러리 및 지원 코드가 제공됩니다.

Open source

오픈 소스 개발 프로세스는 강력하고 안정적인 소프트웨어를 개발하는 가장 신뢰할 수 있는 방법임이 입증되었습니다. 그렇기 때문에 전체 BeeWare 도구 제품군은 BSD 라이선스가 적용되어 누구나 사용하고 수정할 수 있습니다.

All contributions welcome

하지만 코드만 중요한 것이 아닙니다. 성공적인 소프트웨어 프로젝트에는 문서화, 디자인 기술, 피드백 및 버그 보고서가 필요합니다. BeeWare 커뮤니티는 GitHub에 풀 리퀘스트로 들어오는 기여뿐만 아니라 모든 기여가 중요하다는 것을 인정합니다.

From diversity comes strength

다양한 커뮤니티는 강력한 커뮤니티입니다. 이는 모든 수준의 경험, 모든 배경, 모든 인종, 신념, 성향 및 표현을 가진 사람들을 받아들이는 것을 의미합니다. BeeWare 프로젝트는 다양하고 환영하는 커뮤니티를 개발하고 유지하기 위해 최선을 다하고 있습니다. 엄격하게 시행되는 행동 강령을 바탕으로 기여자로 참여하고자 하는 모든 사람에게 공개적으로 멘토링을 제공할 수 있습니다.

기여하세요!

이 모든 것이 흥미롭게 들린다면, BeeWare 프로젝트를 살펴보고 Python 애플리케이션을 개발할 때 어떻게 활용할 수 있는지 알아보는 것은 어떨까요? 또는 기여하고 싶지만 오픈소스에 기여해 본 적이 없다면 처음 기여자를 위한 가이드부터 시작하세요. 바로 시작할 준비가 되었다면 프로젝트 페이지 또는 도움을 줄 수 있는 방법 가이드를 확인하세요.

Contribute to BeeWare

Stay in touch!

마스토돈에서 @beeware@fosstodon.org를 팔로우하거나 BeeWare 매니아 메일링 리스트에 가입하여 프로젝트와 관련된 업데이트, 힌트, 팁, 공지사항을 받아보세요.

계속 연락주세요!
