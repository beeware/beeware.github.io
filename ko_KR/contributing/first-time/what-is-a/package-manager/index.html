패키지 관리자

설치된 패키지를 관리하는 방법

컴퓨터에 소프트웨어를 설치하는 일은 흥미로울 수 있습니다. 때로는 파일을 다운로드한 다음 직접 설치하거나 컴퓨터의 특정 위치에 컴퓨터의 특정 위치에 복사해야 할 때도 있습니다.

하지만 패키지 관리자는 다음과 같은 기능을 통해 이 프로세스를 더 쉽게 수행할 수 있도록 도와줍니다. 소프트웨어 설치 자동화를 지원합니다.

패키지 관리자에는 운영 체제용 패키지 관리자 수준과 파이썬 패키지 전용 패키지 관리자가 있습니다.

macOS

Homebrew는 Mac에 소프트웨어를 설치하기 위한 표준입니다. 이것은 brew install 명령을 실행할 때 사용되는 시스템입니다. 명령을 실행할 때 사용됩니다.

Linux

실행 중인 운영 체제 제품군에 따라 다음을 사용합니다. 'apt-get(Debian 및 Ubuntu용) 또는 yum(Red Hat 및 CentOS용).

파이썬

pip는 파이썬 소프트웨어를 설치하는 방법입니다. pip 설치를 실행하면 는 파이썬 패키징 인덱스, 일명 "PyPI". PyPI는 파이썬 코드의 중앙 저장소입니다. 많은 BeeWare 프로젝트는 pip을 사용하여 설치할 수 있습니다.
