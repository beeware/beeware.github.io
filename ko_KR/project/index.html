유틸리티

그 자체로 유용할 수 있는 다른 BeeWare 도구에서 사용하는 조각을 포함합니다:

Rubicon Objective-C

파이썬과 Objective-C 언어 사이의 브리징을 위한 라이브러리 런타임.

Python Apple Support

macOS, iOS, watchOS 프로젝트에 임베드할 수 있는 파이썬 버전을 구축하기 위한 를 빌드하기 위한 메타 패키지입니다.

... and others

BeeWare는 단일 도구나 라이브러리가 아니라 Python 소프트웨어를 개발, 디버깅, 실행하는 데 사용할 수 있는 프로젝트 의 모음으로, Python 소프트웨어를 개발, 디버그 및 실행하는 데 사용할 수 있습니다. 각 도구는 한 가지 일을 잘 수행한다는 유닉스 철학을 따릅니다. 각 도구 는 단독으로 사용하거나 서로 연결되어 풍부한 프로그래밍 도구 세트를 제공할 수 있습니다.

다락방에는 더 이상 관리하지 않는 프로젝트를 보관하는 다락방(/프로젝트/다락방)]도 있습니다. 더 이상 유지 관리하지 않는 프로젝트를 보관하는 다락방도 있습니다.
