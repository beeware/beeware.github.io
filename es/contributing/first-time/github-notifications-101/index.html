Notificaciones de GitHub 101

Así que ahora usas GitHub y te lo notifica. ¿Y ahora qué?

Una vez que comience a contribuir a BeeWare, GitHub va a empezar a a notificarte cosas.

Esto está muy bien. Pero puede resultar abrumador. consejos para ayudarle a combatir el diluvio.

  • Si prefiere recibir notificaciones sólo por correo electrónico, desactive las notificaciones "Web". notificaciones.
  • Si prefiere no recibir notificaciones por correo electrónico, desactive las notificaciones "Correo electrónico". las notificaciones por correo electrónico.

Si prefiere las notificaciones por correo electrónico, hay un montón de maneras que usted puede manejar las cosas.

Repo settings

Cada repositorio tiene una serie de ajustes de notificación:

  • Observando - recibes notificaciones para __todo__. Cualquier problema, cualquier Pull Request, cualquier comentario. Esto es útil si usted realmente super duper se preocupan por un repo
  • Not Watching - sólo recibirás notificaciones si eres mencionado, o cuando estés participando (por ejemplo, en tu propio Pull Request)
  • Ignorar - nunca jamás serás notificado.

Puedes comprobar qué repos estás vigilando en todo GitHub desde la página (https://github.com/watching).

Filtros de correo electrónico

Los correos electrónicos de GitHub son ricos en información que puede ayudarte a filtrar lo que te interesa y lo que no. lo que te interesa y lo que no.

Los correos electrónicos tendrán el asunto del repositorio en cuestión

  • Por ejemplo, [beeware/batavia] Add a new function sería una notificación del repositorio Batavia

Si te mencionan por tu nombre:

  • el correo electrónico se enviará a mention@noreply.github.com.
  • el correo electrónico incluirá la frase Recibes esto porque has sido mencionado.

Si sólo estás suscrito a un repositorio:

  • el correo electrónico se enviará a subscribed@noreply.github.com.

Ssshhhhhhhhhh

Si quieres silenciar un tema específico, puedes hacer clic en el botón 'Unsubscribe en cualquier tema o pull request.

Si recibes un correo electrónico, puedes hacer clic en "Silenciar este hilo".
