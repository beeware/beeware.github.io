Código de conducta

All participants in the BeeWare community are expected to adhere to a Code of Conduct

Como colaboradores y mantenedores de este proyecto, y en aras de comunidad abierta y acogedora, nos comprometemos a respetar a todas las personas que las personas que contribuyan informando de problemas actualización de la documentación, envío de pull requests o parches, y otras actividades.

Nos comprometemos a que la participación en este proyecto sea experiencia libre de acoso para todos, independientemente de su experiencia, género, identidad y expresión de género, orientación sexual discapacidad, apariencia personal, tamaño corporal, raza, etnia, edad, religión o nacionalidad.

Entre los ejemplos de comportamiento inaceptable por parte de los participantes se incluyen:

  • El uso de lenguaje o imágenes sexualizados
  • Ataques personales
  • Trolling o comentarios insultantes o despectivos
  • Acoso público o privado
  • Publicar información privada de otras personas, como direcciones sin permiso explícito
  • Otras conductas poco éticas o poco profesionales

Los responsables del proyecto tienen el derecho y la responsabilidad de eliminar, editar, o rechazar comentarios, confirmaciones, código, ediciones wiki, problemas y otras contribuciones que no se ajusten a este Código de Conducta, o de expulsar temporal o permanentemente a cualquier colaborador por otros comportamientos que consideren inapropiados, amenazantes, ofensivos o perjudiciales.

Al adoptar este Código de Conducta, los responsables del mantenimiento del proyecto se comprometen a a aplicar estos principios de forma justa y coherente en todos los aspectos de la gestión de este proyecto. Los mantenedores del proyecto que no sigan o hagan cumplir el Código de Conducta podrán ser expulsados permanentemente del equipo del proyecto.

Este código de conducta se aplica tanto en los espacios del proyecto como en los espacios públicos cuando una persona representa al proyecto o a su comunidad. cuando una persona representa al proyecto o a su comunidad.

If you feel the code of conduct has been violated, please report the incident to the BeeWare core team.
