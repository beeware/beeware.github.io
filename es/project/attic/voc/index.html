VOC

Un transpilador que convierte el código de bytes de Python en código de bytes de Java.

El COV cumple dos funciones principales:

  • Compila archivos fuente Python 3.4+ en archivos de clase Java, permitiéndote ejecutar código Python en una JVM (incluida la VM de Android).
  • Proporciona una API que permite crear archivos de clase Java mediante programación.

Todavía no es una implementación completamente compatible con Python 3.4. hay algunas características del lenguaje (como nonlocal y algunas funciones incorporadas) que aún necesitan ser implementadas, y sólo hay una implementación básica de la biblioteca estándar. Sin embargo, es posible convertir programas Python sencillos, e incluso escribir aplicaciones Android Android.

Para probar VOC, consulte la guía Getting Introducción y, a continuación empezar con el primer tutorial.

Si desea contribuir al desarrollo de VOC, disponemos de una guía para primera vez contribuidores.
