Librerías

Colecciones reutilizables de código

Colosseum

Una implementación (parcial) de la caja CSS y el algoritmo de distribución flexbox.

Toga

Un toolkit gráfico nativo de Python y nativo del sistema operativo.

travertino

A base library for building layout algorithms

Ouroboros

Una implementación de la biblioteca estándar de Python, escrita en Python. Esto permite que las librerías estándar de Python se puedan utilicen de forma portátil en cualquier implementación de la máquina virtual Python, es decir, no requiere CPython.

Las librerías son bloques reutilizables de código que se pueden utilizar en un proyecto. Generalmente, las librerías se instalan como dependencias de herramientas, aplicaciones u otras librerías.
