Python Android Template

Una plantilla cookiecutter para crear un proyecto de Android que ejecute código Python.

Python tvOS Template

Una plantilla para la creación de aplicaciones Python que se ejecutarán en dispositivos Apple tvOS.

Python iOS Template

Una plantilla para crear aplicaciones en Python que se ejecutarán en dispositivos iOS.

Python macOS Template

Una plantilla para la creación de aplicaciones Python que se ejecutará en los dispositivos Apple MacOS.

Las plantillas de configuración de Cookiecutter permiten generar el código base necesario para obtener un proyecto de Python que se ejecuta como una aplicación en una plataforma en particular.
