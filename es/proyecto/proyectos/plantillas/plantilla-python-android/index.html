Python Android Template

Una plantilla cookiecutter para crear un proyecto de Android que ejecute código Python.

El contenido de esta página está incompleto, puedes ayudar expandiéndolo.

Este proyecto es una plantilla CookieCutter para un proyecto tipo.

Una plantilla para crear aplicaciones Python que se ejecutarán en dispositivos Android, como teléfonos y tablets. Utiliza cookiecutter que crea proyectos de cookiecutters (plantillas de proyecto).

La forma más fácil de usar este proyecto es no usarlo en absoluto - al menos, no directamente. Briefcase es una herramienta que utiliza esta plantilla, utilizando datos extraídos de tu setup.py.

Las siguientes versiones de Python son compatibles:

  • Python 3.4
  • Python 3.5
