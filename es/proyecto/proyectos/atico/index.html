El ático

No todos pueden ser cantantes...

Batavia

Una implementación en Javascript de la máquina virtual de Python

Python tvOS Support

Un meta-paquete para construir una versión de Python que se puede incrustar en un proyecto de tvOS.

Sealang

Un conjunto extendido de enlaces de Python para libclang

Seasnake

Una herramienta para gestionar la conversión de código C ++ a Python.

VOC

Un transpilador que convierte el bytecode de Python en bytecode de Java.

Waggle

Prepara y carga imágenes de Docker para ser utilizadas por BeeKeeper.

Trebuchet

Una herramienta para implementar sitios web

Cassowary

Una implementación en Python del algoritmo de resolución de restricciones de Cassowary.

cookiecutter-project

Una plantilla cookiecutter para proyectos

cookiecutter-website

Una plantilla cookiecutter para sitios web

Python iOS Support

Un meta-paquete para construir una versión de Python que se puede incrustar en un proyecto de iOS.

Python OSX support

Un meta-paquete para construir una versión de Python que se puede incrustar en un proyecto OS X.

El ático es una colección de proyectos antiguos y experimentales que alguna vez formaron parte de la suite BeeWare, pero que ya no se mantienen activamente. Si estás interesado en asumir el mantenimiento de un proyecto en el ático, por favor ponerse en contacto.
