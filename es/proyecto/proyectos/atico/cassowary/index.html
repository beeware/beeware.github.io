Cassowary

Una implementación en Python del algoritmo de resolución de restricciones de Cassowary.

Este proyecto esta clasificado como en el ático. Ya no se mantiene activamente

Una implementación en Python del algoritmo de resolución de restricciones de Cassowary. El algoritmo de Cassowary forma el núcleo del mecanismo de disposición visual de OSX y iOS.

Este proyecto ya no está en uso. Originalmente fue desarrollado para su uso en Toga; sin embargo, Toga ahora utiliza un mecanismo de disposición visual basado en CSS, implementado a través de Colosseum.
