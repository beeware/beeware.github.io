Herramientas

Proyecto para ayudar al desarrollo

Beefore

Realiza chequeos (como el estilo del código) antes de emerger un Pull Request

Duvet

Herramienta de visualización de cobertura de código

BeeKeeper

Herramienta de construcción y de integración continua

Briefcase

Convert a Python project into a standalone native application.

Cricket

Herramienta gráfica para ejecutar pruebas y encontrar fallas rápidamente

Bugjar

Un depurador gráfico e interactivo para Python.

Galley

Una herramienta gráfica para revisar la documentación de RST / Sphinx

Una herramienta es una clase especial de aplicación que se instala en un entorno virtual de Python. Las herramientas se utilizan para ayudar en el desarrollar de aplicaciones, en funciones de depuración y pruebas hasta empaquetado.
