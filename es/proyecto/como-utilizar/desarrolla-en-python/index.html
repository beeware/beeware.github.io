BeeWare no es sólo un conjunto de herramientas para ayudarle a ejecutar código Python en diferentes plataformas; también es una colección de herramientas para ayudarte a escribir, depurar y afinar tu código Python.

Cricket es una herramienta que ayuda a los proyectos con grandes ensayos de pruebas a identificar los fallos sin esperar a que finalicen todas sus pruebas.

Duvet es una herramienta para mostrar los datos de cobertura de una suite prueba.

Bugjar es un depurador gráfico.

Galley es una herramienta que te permite visualizar fácilmente la documentación de Sphinx mientras la escribes.

Podium es una herramienta de presentación de diapositivas que puedes usar para contar a otros sobre tu proyecto.
