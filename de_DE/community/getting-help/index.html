Hilfe holen

Hängen Sie bei der Verwendung eines BeeWare-Tools fest? Hier erhalten Sie Hilfe.

Wir haben eine Reihe von Kommunikationskanälen, falls Sie mit uns in Kontakt treten möchten.

Echtzeit-Chat

Wir verwenden Discord für die Kommunikation in Echtzeit.

Diskussion und Fehler

A lot of project-specific communication happens on the GitHub Issues for each project. Links for our high-profile projects can be found on our documentation page; visit the GitHub page for the BeeWare organization for the full list of repositories.

Wir nutzen auch GitHub Discussions als Forum, um Projekte zu besprechen. Wenn Sie eine Frage zur Verwendung eines BeeWare-Tools haben, eröffnen Sie einen Diskussions- Thread im Repository für dieses Tool.

Direkte Fragen an die Hauptbeteiligten

Wenn Sie direkte Hilfe benötigen und sich nicht wohl dabei fühlen, ein öffentliches Forum zu benutzen, können Sie sich gerne an einen Core Contributor, entweder per E-Mail, Twitter, Mastodon oder Discord.

Beachten Sie jedoch, dass Ihre Frage in einem öffentlichen Forum gestellt werden kann. Forum gestellt werden kann, sollte sie das auch. Dadurch wird sichergestellt, dass so viele Menschen wie möglich von einer Antwort profitieren können - wenn Sie ein Problem haben Wenn Sie ein Problem haben, ist es sehr wahrscheinlich, dass jemand anderes das gleiche Problem hat.

Darüber hinaus bietet das Kernteam keine kostenlose Unterstützung oder Fehlerbehebung Hilfe für kommerzielle Projekte. Wenn Sie nicht in der Lage sind, Ihre Frage in einem öffentlichen Forum zu stellen, müssen Sie jemanden suchen, der bereit ist, Ihnen kommerziellen Support für Ihre Bedürfnisse zu finden.
