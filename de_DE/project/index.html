BeeWare ist kein einzelnes Werkzeug oder eine Bibliothek - es ist eine Sammlung von Projekten die zur Entwicklung, Fehlersuche und Inbetriebnahme von Python-Software verwendet werden können. Jedes Werkzeug folgt der Unix-Philosophie, eine Sache gut zu machen. Jedes Werkzeug kann isoliert verwendet werden, oder sie können miteinander verkettet werden, um einen reichhaltigen Satz von Programmierwerkzeugen.

Es können aber nicht alle Knaller sein - es gibt auch einen Dachboden, auf dem wir Projekte aufbewahren, die wir nicht mehr betreuen.
