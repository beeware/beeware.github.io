Briefcase

Wandle deine Python Projekte zu eigenständige Apps um.

Briefcase ist ein Werkzeug um Python Projekte zu eigenständige Apps umzuwandeln. Man kann Projekte für viele verschiedene Platformen erstellen:

  • Mac
  • Windows
  • Linux
  • iPhone/iPad
  • Android

Unterstützung für AppleTV, watchOS, wearOS und web applikationen ist geplannt.

Falls du Briefcase in der Praxis sehen willst, folge den BeeWare tutorial. Dieser Tutorial zeigt dir den Weg um neue Apps mit Briefcase zu kreieren aber auch zu verpacken.

How to help

Briefcase ist Teil der Beeware-Suite.

Wenn du Probleme mit Briefcase hast, logge dich auf GitHub ein. Wenn du Code beitragen willst, `forke`_ bitte den Code und sende eine `Pull-Anfrage`_.
