Duvet

Coverage visualization tool

This page's contents is incomplete. You can help by expanding it.

Duvet is a GUI tool for visualizing code coverage results produced by coverage.py.

So, why is it called "Duvet"?

Because a duvet doesn't just provide coverage - it keeps you warm and cozy.
