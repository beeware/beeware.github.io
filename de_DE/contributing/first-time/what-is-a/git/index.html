Was ist Git überhaupt?

git ist ein Version Control System, mit dem wir Änderungen am Code im Laufe der Zeit zu speichern und weiterzugeben.

GitHub ist unter der Haube einfach git, enthält aber eine Menge hilfreicher Dinge, wie eine Web-Schnittstelle und nette Pull Requests und Issue Tracking Systeme.
