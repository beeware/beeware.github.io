VOC

Ein Transpiler, der Python-Bytecode in Java-Bytecode umwandelt.

VOC erfüllt zwei wichtige Funktionen:

  • Kompiliert Python 3.4+ Quelldateien in Java-Klassendateien, so dass Sie die Ausführung von Python-Code auf einer JVM (einschließlich der VM von Android).
  • Bietet eine API, mit der Sie programmatisch Java-Klassendateien erstellen können.

Es ist noch keine voll konforme Python 3.4 Implementierung - es gibt gibt es einige Sprachmerkmale (wie nonlocal und einige eingebaute Funktionen) die noch implementiert werden müssen, und es gibt nur eine einfache Implementierung der Standardbibliothek. Es ist jedoch möglich, einfache einfache Python-Programme zu konvertieren, und sogar [einfache Android Anwendungen] zu schreiben (https://github.com/eliasdorneles/tictactoe-voc).

Um VOC auszuprobieren, gehen Sie die Anleitung Erste Schritte Erste Schritte durch, dann beginnen Sie mit dem ersten Lernprogramm.

Wenn Sie zur VOC-Entwicklung beitragen möchten, haben wir einen [Leitfaden für ersten Mal Beitragende] (https://voc.readthedocs.io/en/latest/how-to/contribute-code.html).

So, why is it called "VOC"?

VOC ist ein Backronym für "Vestigial Output Compiler". Oder "Vexing Obtuse Compiler". Oder "Wertvoller obskurer Compiler". Oder "Variegated Ocelot Kalibrator". Sie haben die Wahl.

Historisch gesehen war die VOC die Vereenigde Oostindische Compagnie (VOC), oder die Niederländische Ostindien-Kompanie, die oft als das erste multinationale multinationales Unternehmen. Sie war auch das erste Unternehmen, das Aktien ausgab und den Handel mit diesen Aktien erleichterte. Sie erhielt ein 21 Jahr das Monopol für den Handel in Asien, vor allem auf den Gewürzinseln Gewürzinseln - den Niederländischen Ostindien. Sie errichteten einen großen Handelshafen in Batavia - dem heutigen Jakarta - auf der Insel Java (heute Teil von Indonesien).

Während die VOC für ihre Aktionäre ein sehr profitables Unternehmen war, agierte sie operierte sie als Kolonialmacht. Moderne Historiker haben [sich Kritik an ihren Geschäfts Geschäftspraktiken] (https://en.wikipedia.org/wiki/Dutch_East_India_Company#Criticism), die die Ausbeutung der einheimischen Bevölkerung (einschließlich des Einsatzes von Sklavenarbeit), Sklavenhandel, Gewaltanwendung und Umweltzerstörung Zerstörung der Umwelt (einschließlich Abholzung).
