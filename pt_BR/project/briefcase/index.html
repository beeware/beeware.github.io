Briefcase

Converta um projeto Python em um aplicativo nativo autônomo.

O Briefcase é uma ferramenta para converter um projeto Python em um aplicativo nativo autônomo. aplicativo nativo autônomo. Você pode empacotar projetos para:

  • macOS (como um pacote .app, arquivo DMG ou instalador PKG)
  • Windows (como um arquivo .zip ou um instalador MSI)
  • Linux (como um Flatpak ou um pacote nativo do sistema .rpm, .deb ou .pkg.zip) nativo do sistema)
  • iPhone/iPad
  • Android
  • Web

O suporte para Apple TV, watchOS e Wear OS está planejado.

Se você quiser ver o Briefcase em ação, experimente o [BeeWare tutorial] (https://beeware.readthedocs.io). Esse tutorial orienta você pelo processo de criação e empacotamento de um novo aplicativo com o Briefcase.

Como ajudar

O Briefcase faz parte do pacote BeeWare.

Se tiver problemas com o Briefcase, registre-os em GitHub. Se você quiser contribuir com o código, faça um fork do código e envie uma pull request.
