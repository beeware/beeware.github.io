VOC

Um transpilador que converte bytecode Python em bytecode Java.

O VOC tem duas funções principais:

  • Compila arquivos de origem Python 3.4+ em arquivos de classe Java, permitindo que você executar o código Python em uma JVM (incluindo a VM do Android).
  • Fornece uma API para permitir que você crie arquivos de classe Java de forma programática.

Ainda não é uma implementação completamente compatível com o Python 3.4 - há há alguns recursos da linguagem (como nonlocal e algumas funções embutidas) que ainda precisam ser implementadas, e há apenas uma implementação da biblioteca padrão. No entanto, é possível converter programas Python simples e até mesmo escrever [aplicativos Android simples] (https://github.com/eliasdorneles/tictactoe-voc).

Para dar uma olhada no VOC, leia o guia [Getting Started] (https://voc.readthedocs.io/en/latest/index.html) e, em seguida comece com o [primeiro tutorial] (https://voc.readthedocs.io/en/latest/tutorial/tutorial-0.html).

Se você quiser contribuir com o desenvolvimento do VOC, temos um [guia para primeira vez contributors] (https://voc.readthedocs.io/en/latest/how-to/contribute-code.html).

Então, por que é chamado "VOC"?

VOC é um backronym para "Vestigial Output Compiler" (compilador de saída vestigial). Ou "Vexing Obtuse Compilador". Ou "Valuable Obscure Compiler" (compilador obscuro valioso). Ou "Variegated Ocelot Calibrador". A escolha é sua.

Historicamente, VOC era a Vereenigde Oostindische Compagnie (VOC), ou Companhia Holandesa das Companhia Holandesa das Índias Orientais, geralmente considerada a primeira corporação multinacional do mundo. Ela também foi a primeira empresa a emitir ações e a facilitar a negociação dessas ações. Foi concedido a ela um monopólio de 21 monopólio de 21 anos para realizar atividades comerciais na Ásia, principalmente nas Ilhas das Especiarias - as Índias Orientais Holandesas. Eles estabeleceram um importante porto comercial em Batávia, hoje Jacarta, na ilha de Java (hoje parte da Indonésia). Indonésia).

Embora a VOC fosse um negócio muito lucrativo para seus acionistas, ela operava como uma potência colonial. Os historiadores modernos [expressaram críticas às suas práticas comerciais práticas comerciais] (https://en.wikipedia.org/wiki/Dutch_East_India_Company#Criticism), que incluíam a exploração de populações indígenas (incluindo o uso de trabalho escravo), comércio de escravos, uso de violência e destruição destruição ambiental (incluindo o desmatamento).

O VOC também desempenhou um papel importante na descoberta da Austrália pelos europeus, e há Há muitos naufrágios de navios do VOC na costa do oeste da Austrália.

O nome VOC foi escolhido pelo projeto porque o fundador do projeto é é da Austrália Ocidental, e VOC tem uma forte presença não colonial na história história desse estado. Também foi um trocadilho conveniente, pois a VOC era uma empresa de Java empresarial.
