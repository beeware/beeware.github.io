Além da Web 2.0 - Django e Python no ecossistema moderno da Web

por Russell Keith-Magee; publicado 1 April 2016

O Django tem mais de 10 anos, mas a Web para a qual ele foi criado - o mundo da "Web 2.0" - não existe mais. Sim, ainda precisamos desenvolver rapidamente sites com base em banco de dados, habilitados para AJAX, mas a mas a Web moderna está enfrentando novos problemas e novos desafios. Muitos desses desafios envolvem a interação com dispositivos que não são desktops e plataformas que não são um navegador tradicional - lugares onde o Python não é um cidadão de primeira classe.

Isso significa que temos que abandonar o Django e o Python em favor de outras estruturas e linguagens? Podemos continuar usando Python e Django no lado do servidor e interagir com estruturas independentes do lado do cliente? Ou podemos empurrar o Python e Django para esses novos ambientes?

Nesta palestra, o Dr. Russell Keith-Magee explorará os problemas e as e as possíveis soluções para esses problemas.

Python em movimento: o estado do Python móvel

por Russell Keith-Magee; publicado 2 August 2015

Nos últimos 5 a 10 anos, a divisão "desktop/servidor" que dominou a a computação nos anos 80, 90 e início dos anos 2000, foi acompanhada por um terceiro dispositivos móveis, como telefones e tablets.

Então, qual é a história para executar o Python nesses dispositivos? Embora não tenha havido muito o que dizer no passado, a boa notícia é que, no ano passado, houve algumas mudanças significativas no cenário, e o Python em dispositivos móveis está agora à beira de se tornar uma verdadeira realidade.

Nesta palestra, o Dr. Russell Keith-Magee fará um levantamento do estado atual do uso do Python em dispositivos móveis, o trabalho que está sendo feito atualmente, áreas em que é necessário mais trabalho e as perspectivas para o futuro.
