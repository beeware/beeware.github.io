Além da Web 2.0 - Django e Python no ecossistema moderno da web

por Russell Keith-Magee; publicado 1 April 2016

Django tem mais de 10 anos de idade; mas a web em que ele foi construído - o mundo da "Web 2.0" - não existe mais. Sim, ainda precisamos desenvolver rapidamente baseados em banco de dados, com AJAX disponível, porém, a web moderna enfrenta novos problemas e desafios. Muitos desses desafios envolvem interação com dispositivos que não são máquinas desktop e plataformas que não utilizam um navegador tradicional - lugares onde o Python não é um cidadão de primeira-classe.

Isto significa que temos que abandonar Django e Python para usar outras linguagens e frameworks? Podemos continuar utilizando Python e Django do lado do servidor para interagir com frameworks independentes do lado do cliente? Ou podemos utilizar Python and Django nesses novos ambientes?

Nessa palestra, Dr. Russell Keith-Magee explora os problemas e as possíveis soluções para eles.

Como visto em DjangoCon Europe 2016

Python em movimento: O estado do Python móvel

por Russell Keith-Magee; publicado 2 August 2015

Nos últimos 5-10 anos a separação "desktop/servidor" que dominou a computação nos anos 80, 90 e no início dos anos 2000 foi jogada para o terceiro plano - devido os dispositivos móveis como smartphones e tablets.

Então, qual é a história do Python nesses dispositivos? Embora não se tenha muito a dizer do passado, a boa notícia é que no último ano houve mudanças significativas no horizonte e agora Python no Mobile está prestes a se tornar uma realidade.

Nessa palestra, Dr. Russell Keith-Magee vai responder o estado da arte de usar Python nos dispositivos móveis, o trabalho que está sendo feito, as áreas onde mais trabalho é necessário e as perspectivas do futuro.
