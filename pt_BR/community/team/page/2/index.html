O Bee Team

O time central por trás do desenvolvimento do BeeWare

Quem são as pessoas por trás do BeeWare? Existe um enorme grupo de contribuidores, mas o projeto é administrado pelo Bee Team.

Ele também é o fundador do projeto BeeWare, desenvolvendo GUI tools para dar suporte ao desenvolvimento de software Python.
