Com mais de 45 milhões de usuários, o [Anaconda] (https://anaconda.com) é o sistema operacional mais popular para IA, fornecendo acesso aos pacotes Python de código aberto fundamentais usados em IA moderna, ciência de dados e aprendizado de máquina por meio de uma plataforma integrada. O Anaconda foi pioneiro no uso pioneiro no uso do Python para ciência de dados, defendeu sua vibrante comunidade e continua a administrar projetos de código aberto que tornam possíveis as inovações do futuro. Suas soluções de nível empresarial permitem que as empresas, de pesquisa e acadêmicas em todo o mundo para aproveitar o poder do código aberto para obter vantagens competitivas, pesquisas inovadoras e um mundo melhor.
