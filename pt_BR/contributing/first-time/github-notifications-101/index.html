Notificações do GitHub 101

Então, agora você está usando o GitHub e ele está notificando você. E agora?

Quando você começar a contribuir com o BeeWare, o GitHub começará a a notificá-lo sobre as coisas.

Isso é muito bom! Mas pode se tornar esmagadora; portanto, aqui estão algumas dicas para ajudá-lo a combater o dilúvio.

  • Se você preferir apenas notificações por e-mail, desative as notificações "Web". notificações.
  • Se você preferir não receber notificações por e-mail, desative as notificações de "E-mail". notificações.

Se você preferir as notificações por e-mail, há várias maneiras de lidar com as coisas.

Configurações de repo

Cada repositório tem várias configurações de notificação:

  • Observando - você recebe notificações sobre __everything___. Qualquer problema, qualquer Pull Request, qualquer comentário. Isso é útil se você realmente super duper se importa com um repositório
  • Not Watching - você só receberá notificações se for mencionado ou quando estiver participando (por exemplo, em sua própria solicitação pull)
  • Ignorar - nunca, jamais, será notificado.

É possível verificar quais repositórios você está observando em todo o GitHub na página [watching page] (https://github.com/watching).

Filtros de e-mail

Os e-mails do GitHub são ricos em informações que podem ajudá-lo a filtrar o que o que lhe interessa e o que não lhe interessa.

Os e-mails terão a linha de assunto do repositório em questão

  • Por exemplo, [beeware/batavia] Add a new function seria uma notificação do repositório Batavia

Se você for mencionado pelo nome:

  • o e-mail será enviado em cópia para mention@noreply.github.com
  • o e-mail incluirá a frase Você está recebendo este e-mail porque foi mencionado.

Se você estiver inscrito apenas em um repositório:

  • O e-mail será enviado em cópia para subscribed@noreply.github.com

Ssshhhhhhhhhhhh

Se quiser silenciar um tópico específico, você pode clicar no botão "Cancelar inscrição em qualquer problema ou pull request.

Se você receber um e-mail, poderá clicar em "Silenciar este tópico".
