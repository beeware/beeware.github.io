Síndrome do Impostor

Você acha que não está pronto para ser um colaborador de código aberto? Você está errado.

Pode haver uma vozinha dentro de sua cabeça dizendo que você não está pronto, que precisa fazer mais um você não está pronto; que precisa fazer mais um tutorial; que não está pronto para ser um colaborador de código aberto. Afinal de contas, você é apenas um iniciante. O que você poderia oferecer a um projeto como o BeeWare?

Eu lhe garanto: a vozinha em sua cabeça está errada. Se você sabe escrever código, você pode contribuir com código para o código aberto e para o BeeWare.

Você também não é a primeira pessoa a ter esses pensamentos. Até mesmo os membros da equipe principal deste projeto têm esses pensamentos de vez em quando. de vez em quando. Isso é chamado de "síndrome do impostor" e é um problema muito comum. comum. A boa notícia é que estamos aqui para ajudá-lo a superar isso.

Este tutorial existe para garantir que você saiba exatamente o processo que deve a seguir para que seu patch seja mesclado. Além dessas instruções de procedimento, este projeto tem um Código de Conduta. Todo mundo comete erros - isso é uma parte natural do aprendizado. Nosso compromisso com você é que estamos aqui para ajudá-lo a aprender, não para insultá-lo ou menosprezá-lo por aprender.

Ser um colaborador de código aberto também não significa apenas escrever código. Você pode ajudar escrevendo documentação, testes ou até mesmo dando feedback sobre o projeto (e sim, isso inclui dar feedback sobre o processo de contribuição). sobre o processo de contribuição). Algumas dessas contribuições podem ser as mais valiosas para o projeto como um todo, porque você está chegando ao projeto com novos olhos, de modo que você pode ver os erros e as suposições que os que os colaboradores mais experientes não perceberam.

Você também não pode causar nenhum dano, seja ao seu próprio computador ou ao projeto como um todo. Os projetos do BeeWare não tocam em nenhuma parte de seu computador ou sistema operacional que possa causar danos graves. Portanto, não tenha medo de contribuir. Se você chegou até aqui, você demonstrou que tem interesse em contribuir - e isso é tudo o que você precisa. Podemos ajudá-lo no restante do caminho.
