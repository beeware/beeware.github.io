行為守則

All participants in the BeeWare community are expected to adhere to a Code of Conduct

身為本專案的貢獻者與維護者，並為了促進開放且友善的社群，我們承諾尊重所有的人。 我們承諾尊重所有透過回報問題、張貼功能表 我們保證尊重所有透過報告問題、張貼功能 請求、更新文件、提交拉取請求或修補程式、 及其他活動做出貢獻的人。

我們致力於讓每個人在參與此專案時 騷擾的體驗，不論其經驗、性別、性別認同和表現、性取向 經驗、性別、性別認同和表現、性取向、殘障、個人外貌、體型、種族、民族、年齡、膚色、性傾向 殘障、個人外貌、體型、種族、民族、年齡、宗教或國籍、 宗教或國籍。

參與者不可接受的行為包括：

  • 使用性化語言或圖像
  • 人身攻擊
  • 釣魚或侮辱/詆毀性的評論
  • 公開或私下騷擾
  • 未經明確允許，公開他人的私人資訊，例如物理或電子 未經明確允許的地址
  • 其他不道德或不專業的行為

專案維護者有權利和責任移除、編輯、 或拒絕不符合本行為準則的評論、提交、程式碼、維基編輯、問題和其他貢獻。 不符合本行為準則的貢獻，或暫時或永久禁止任何貢獻者的其他不符合本行為準則的行為。 暫時或永久禁止任何貢獻者的其他行為。 他們認為不適當、具威脅性、攻擊性或有害的行為。

採用本行為準則，專案維護者承諾 公平、貫徹地將這些原則應用到管理本專案的每一方面。 管理這個專案。專案維護人員如果不遵守或執行 行為準則的專案維護人員，可能會被永久地從專案團隊中除名。

本行為守則適用於專案區域內，以及個人代表專案或其社區時的公共區域。 當個人代表專案或其社區時，本行為守則也適用於專案空間和公共場所。

If you feel the code of conduct has been violated, please report the incident to the BeeWare core team.
