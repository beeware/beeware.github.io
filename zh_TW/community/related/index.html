Chaquopy

Chaquopy是一套適用於 Android 的 Python SDK，可與 Android Studio 的標準 Gradle 建置系統整合。 與 Android Studio 的標準 Gradle 建立系統整合、 與 Java/Kotlin 互通的簡單 API，以及各種 第三方 Python 套件。它被用來作為 BeeWare Android 工具的基礎。
