移動中的 Python：行動 Python 的現況

By Russell Keith-Magee; published 8月 2, 2015

在過去 5 到 10 年間，在 80、90 年代和 2000 年初主宰運算的「桌上型電腦/伺服器」分離模式，現在已加入第三個分離模式。 在過去 5 到 10 年間，在 80、90 和 21 世紀初期主宰電腦運算的 行動裝置，例如手機和平板電腦。

那麼，在這些裝置上執行 Python 有什麼故事呢？雖然 在這些裝置上執行 Python 的情況如何？ 在過去的一年裡，Python 的面貌發生了一些顯著的變化、 在移動設備上運行 Python 現在已經瀕臨真正的 現實。

在這場演講中，Russell Keith-Magee 博士將探討在行動裝置上使用 Python 的現況、目前所做的工作，以及如何在行動裝置上使用 Python。 目前正在進行的工作、 需要進行更多工作的領域，以及未來的展望。

As seen at PyCon AU 2015
