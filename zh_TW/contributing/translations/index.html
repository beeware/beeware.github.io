本網站

想要協助翻譯或更新本網站的內容嗎？

這些頁面的內容大多是以 Lektor (.lr) 檔案 和 .ini 資料包。要 要翻譯頁面，請使用右上方的地球儀選擇所需的語言，然後點選 然後按一下「在 GitHub 上編輯」。若要翻譯資料包，請 開啟 .ini 檔案 並建立或編輯指定語言的部分 (例如，[pt] 表示葡萄牙文)。 表示葡萄牙語）。

The BeeWare Tutorial

我們也維護 BeeWare 教程的翻譯。這些翻譯是使用 使用 Weblate，這是一個管理翻譯的 web 介面來管理翻譯。如果您想參與 這些翻譯，請在 Weblate上建立一個帳號，然後在 Discord 上加入 在 Discord 上的 #translations 頻道，並告訴我們您的 Weblate 帳戶 使用者名稱。我們會將您加入 Weblate 上的翻譯團隊，您就可以 開始工作！

正在翻譯？在 Discord 上加入 BeeWare 團隊

如果您有興趣協助這些翻譯工作，請考慮 加入 BeeWare [Discord] (https://beeware.org/bee/chat/)。您將 找到 #translations 頻道，在這裡團隊可以回答任何 問題並提供指導。
