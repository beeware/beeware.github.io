BeeWare 並非單一的工具或函式庫 - 它是一系列專案的集合。 可以用來協助開發、除錯及啟動 Python 軟體。 每個工具都遵循 Unix 的哲學：做好一件事。每個工具 可以單獨使用，也可以串連起來，提供一套豐富的程式設計工具。 豐富的程式設計工具集。

不過不可能全都是俏皮話 - 我們還有一個 閣樓](/project/attic/) 存放我們不再維護的專案。
