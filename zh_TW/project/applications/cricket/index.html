Cricket

快速發現故障的測試執行器 GUI

Cricket 是一個圖形工具，可協助具有大型測試套件的專案 無需等待所有測試完成即可找出故障。

一般的 unittest 測試執行程式會將所有輸出轉換到主控台，並且在套件執行時只提供很少的細節。 套件執行時提供的細節很少。因此

  • 在測試套件執行完成之前，您無法開始檢視故障。 才能開始檢視失敗、
  • 對於識別測試失敗的模式來說，這不是一種非常容易使用的格式。 失敗、
