Batavia

Python 虛擬機的 Javascript 實作

Batavia 是以 Javascript 寫成的 Python 虛擬機實作。 Javascript 寫成。使用 Batavia，您可以在瀏覽器中執行 Python 字節碼。

它尊重 Python 3.4.4+ 的語法和慣例，並允許您參照 物件和類別。
