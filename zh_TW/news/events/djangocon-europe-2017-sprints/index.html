DjangoCon Europe 2017

星期四, 四月 6, 2017

BeeWare will be running a sprint at DjangoCon Europe 2017.

Russell Keith-Magee will be at the sprints, and will be more than willing to help First Time Contributors earn their own shiny Challenge Coin .

For more information, check the DjangoCon Europe 2017 website

What is a Sprint?

Code sprints are events held normally after a conference where attendees can use the conference venue to work in small groups on various projects. Learn more about how sprints work with the PyCon US 'What is a sprint?' video
