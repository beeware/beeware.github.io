Uzyskiwanie pomocy

Utknąłeś podczas korzystania z narzędzia BeeWare? Oto jak uzyskać pomoc.

Dysponujemy wieloma kanałami komunikacji, jeśli chcesz się z nami skontaktować.

Czat w czasie rzeczywistym

Używamy Discord do komunikacji w czasie rzeczywistym.

Dyskusja i błędy

A lot of project-specific communication happens on the GitHub Issues for each project. Links for our high-profile projects can be found on our documentation page; visit the GitHub page for the BeeWare organization for the full list of repositories.

We also use GitHub Discussions as a forum to discuss projects. If you have a question about how to use a BeeWare tool, open a discussion thread in the repository for that tool.

Bezpośrednie pytania do głównych współpracowników

Jeśli potrzebujesz bezpośredniej pomocy i nie czujesz się komfortowo korzystając z publicznego forum, prosimy o kontakt z dowolnym Core Contributor, przez e-mail, Twitter, Mastodon lub Discord.

Należy jednak pamiętać, że jeśli pytanie może zostać zadane na publicznym forum forum, należy to zrobić. Zapewni to, że jak najwięcej osób będzie mogło w stanie skorzystać z każdej odpowiedzi, która zostanie udzielona - jeśli masz problem, jest wysoce prawdopodobne, że ktoś inny ma ten sam problem.

Ponadto, główny zespół nie zapewni bezpłatnego wsparcia ani pomocy w debugowaniu dla projektów komercyjnych. Jeśli nie jesteś w stanie opublikować pytania na publicznym forum, będziesz musiał poszukać kogoś chętnego do zapewnienia wsparcie komercyjne, aby zaspokoić swoje potrzeby.
