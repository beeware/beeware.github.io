VOC

Transpiler konwertujący kod bajtowy Pythona na kod bajtowy Javy.

VOC pełni dwie główne funkcje:

  • Kompiluje pliki źródłowe Python 3.4+ do plików klas Java, umożliwiając uruchamianie kodu Pythona na maszynie JVM (w tym na maszynie wirtualnej Androida).
  • Udostępnia interfejs API umożliwiający programowe tworzenie plików klas Java.

Nie jest to jeszcze w pełni zgodna implementacja Pythona 3.4 - istnieją są pewne funkcje językowe (takie jak nonlocal i niektóre wbudowane funkcje) które wciąż muszą zostać zaimplementowane, a istnieje tylko implementacja biblioteki standardowej. biblioteki standardowej. Możliwe jest jednak konwertowanie prostych programów w Pythonie, a nawet pisanie prostych aplikacji na Androida aplikacje.

Aby wypróbować VOC, zapoznaj się z przewodnikiem Getting Started, a następnie rozpocząć od pierwszego tutorial.

Jeśli chcesz przyczynić się do rozwoju VOC, mamy przewodnik dla pierwszy raz contributors.
