Duvet

Werkzeug zur Visualisierung der Abdeckung

Ta strona jest niekompletna. Możesz pomóc uzupełniając ją.

Duvet ist ein GUI Werkzeug zur Visualisierung von Code Abdeckung; Ergebnisse produziert von coverage.py.

Dlaczego więc nazywa się to "Duvet"?

Weil eine Bettdecke nicht nur Abdeckung zur Verfügung stellt, sondern auch Wärme und Gemütlichket. (Duvet bedeutet Bettdecke auf Englisch)
