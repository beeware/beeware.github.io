Ouroboros

A standalone, pure Python implementation of the Python Standard Library. This allows Python default libraries to be used portably on any implementation of the Python virtual machine---not requiring CPython.

A standalone, pure Python implementation of the Python Standard Library.

How to help

Expansion of unit testing.

So, why is it called "Ouroboros"?

The ouroboros is an ancient symbol depicting a serpent eating its own tail. Ouroboros (this library) is also similarly self-referential; it implements Python in Python itself (the subset of Python on the Python virtual machine).
