Beyond Web 2.0 - Django og Python i det moderne webøkosystem

by Russell Keith-Magee; published 1 April 2016

Django er over 10 år gammel; men det web, som det blev bygget til - verdenen af "Web 2.0"-verdenen - findes ikke rigtig mere. Ja, vi har stadig brug for for hurtigt at udvikle databasebaserede hjemmesider med AJAX, men det moderne web står over for nye problemer og nye udfordringer. Mange af disse udfordringer involverer interaktion med enheder, der ikke er stationære maskiner og platforme, der ikke er en traditionel browser - steder, hvor Python ikke er en førsteklasses borger.

Betyder det, at vi skal opgive Django og Python til fordel for andre frameworks og sprog? Kan vi fortsætte med at bruge Python og Django på serversiden og interagere med uafhængige frameworks på klientsiden? Eller kan vi skubbe Python og Django ind i disse nye miljøer?

I dette foredrag vil Dr. Russell Keith-Magee udforske problemerne og de potentielle løsninger på disse problemer.

Python på farten: Status for mobil Python

by Russell Keith-Magee; published 2 August 2015

I løbet af de sidste 5-10 år har den "desktop/server"-opdeling, der dominerede computere i 80'erne, 90'erne og begyndelsen af 2000'erne, fået selskab af en tredje spiller - mobile enheder som telefoner og tablets.

Så hvad er historien om at køre Python på disse enheder? Selvom der ikke har været meget at sige tidligere, er den gode nyhed, at der i løbet af er, at der i løbet af det sidste år er sket nogle markante ændringer i landskabet, og Python på mobilen er nu på nippet til at blive en reel realitet. virkelighed.

I dette foredrag vil Dr. Russell Keith-Magee undersøge den nuværende tilstand af brug af Python på mobile enheder, det arbejde, der i øjeblikket udføres, områder, hvor der er behov for mere arbejde, og udsigterne for fremtiden.
