修补 Tkinter

作者：Russell Keith-Magee; 发布于 七月 10, 2013

Tkinter 是 Tk 图形库的 Python 封装器，从很早以前起就是 Python 标准库的一部分。 从很早开始就是 Python 标准库的一部分。不过 并没有被广泛使用。

这是有充分理由的。Tk 的文档 糟糕。如果你能跨过这道坎，Tkinter 应用程序看起来也很糟糕 它们的小部件数量少得可怜，样式也只有 Tkinter 应用程序看起来糟透了--它们只有一套远远不够用的部件，而且还是用 20 世纪 90 年代中期最先进的开源图形技术设计的。

然后，全世界都沉迷于网络框架，桌面 宣布死亡。

然而，在过去几年中，Tkinter 被忽视的许多原因 已悄然修复。Tk 8.4 大幅改进了 Tk 的视觉外观。 tkdocs.com 的出现，解决了许多 Tk 文档。

在本讲座中，您将重新认识这位老朋友，并了解为什么在网络和移动世界中您应该关注 在网络和移动世界中，您应该关注它的原因。
