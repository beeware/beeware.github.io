Rubicon Objective-C

用于连接 Python 和 Objective-C 语言的运行库 运行时

Rubicon Objective-C 是连接 Objective-C 和 Python 的桥梁。它使您能够：

  • 使用 Python 来实例化 Objective-C 中定义的对象、
  • 使用 Python 在 Objective-C 中定义的对象上调用方法，以及
  • 在 Python 中子类和扩展 Objective-C 类。

它还包括在 Foundation 框架中的一些关键数据类型的封装（例如，NSString.

您可以在 GitHub 上找到 rubicon-objc。
