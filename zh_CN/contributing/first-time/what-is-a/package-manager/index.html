软件包管理器

如何管理已安装的软件包

在电脑上安装软件可能很有趣。有时您 下载文件，然后自己安装，或将文件复制到电脑上的特定位置。 电脑上的特定位置。

不过，软件包管理器允许 自动安装软件。

有不同级别的软件包管理器：操作系统级别的 级别，以及专门针对 Python 软件包的级别。

macOS

Homebrew是在 Mac 上安装软件的标准。 Mac 上安装软件的标准。如果运行 brew install 命令，就会使用该系统。 命令时使用的系统。

Linux

根据您运行的操作系统系列，您可以使用 apt-get"（适用于 Debian 和 Ubuntu）或 "yum"（适用于 Red Hat 和 CentOS）。

Python

pip 是安装 Python 软件的方法。运行 pip install 时 会使用 Python Packaging Index，也称为 "PyPI"。PyPI 是 Python 代码的中心存储库。许多 BeeWare 项目都可以使用 pip 安装。
