翻译

您会多种语言吗？请帮助我们翻译本网站和 BeeWare 教程！

本网站

希望帮助翻译或更新本网站内容？

这些网页的翻译由 Weblate（一款管理翻译的网络界面） 管理。如果您想贡献翻译，请在 Weblate上注册账户，然后加入 #translations 频道，并告诉我们您的 Weblate 用户名。我们将把您添加到 Weblate 上的翻译团队，您就可以开始了！

In addition, the .ini databags contain recurring phrases used on various components on the website. To translate those short phrases, edit them in your fork and then submit a pull request against the lektor branch of beeware/beeware.github.io to modify those translations. Try to keep the context of those appearing phrases in mind; they're used in HTML files in the template directory in that repository.

BeeWare 教程

我们也使用 Weblate 管理 BeeWare 教程 的翻译。像此网站的内容一样，您可以在 Weblate 上注册账户，在 Discord 上介绍自己，然后即可开始翻译。

想帮助翻译？在 Discord 上加入 BeeWare 团队

如果您有兴趣参与这些翻译工作，请考虑加入 BeeWare Discord。您将找到 #translations 频道，团队将在此回答您的任何问题并提供指导。
